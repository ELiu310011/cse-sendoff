Interactive Python Adventure

This project is a fictional choose-your-own-adventure story created through the repl.it user interface. Completed using limited python knowledge, each story is individualized to the different pairs who created them. Our group specifically created a story about a legendary hero on a quest to save a princess. Not too original, but it was a pleasure to code!
