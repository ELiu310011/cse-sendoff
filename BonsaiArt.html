BonsaiJS Art

This work is a static image ( or GIF in some cases ) generated through the creation of several, smaller shapes in order to form a larger product. In my individual case, I created a gif containing rapidly rotating shurikens, or ninja stars, using several triangles set to rotate at the same degree at the same time. This project served as a helpful introduction to this language, and a stellar demonstration of simple shapes and simple code.

var ninja = new Bitmap('https://upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Ninja-kanji.svg/1280px-Ninja-kanji.svg.png', function(err) {
  if (err) return;
  this.attr({
    y: -200,
    x: 900,
    scale: .15,
  });
  ninja.animate('2000ms', { opacity: 0 });
  stage.addChild(this);
  this.animate('1s', {x: 0, y: 0, rotation: 0});
});
